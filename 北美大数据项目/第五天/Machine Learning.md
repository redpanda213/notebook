## Categories of Machine Learning

- Supervised Learning: Predict class from observations 

  监督式学习----一种来自观察结果的预测类型

  - Classification  分类

  - Regression 回归分析

- Unsupervised Learnin(Clustering): group observations into meaningful groups.

  `clustering`聚类的目的在于把相似的东西聚在一起，而我们并不关心这一类是什么。
